# Discovery Feature Module

Location-based discovery and matching.

## Structure

- `components/` - Discovery UI (ProfileCard, FilterPanel, etc.)
- `hooks/` - Discovery hooks (useNearbyFemales, useFilters, etc.)
- `services/` - Discovery API calls
- `types/` - Discovery types (NearbyUser, FilterOptions, etc.)

## Responsibilities

- Location-based user discovery
- Filtering (age, distance, interests)
- Profile card display
- Distance calculation
- Online status display
- Match suggestions
