# Admin Feature Module

Admin panel functionality.

## Structure

- `components/` - Admin UI components
- `hooks/` - Admin hooks
- `services/` - Admin API calls
- `types/` - Admin types
- `pages/` - Admin pages (Dashboard, UserManagement, etc.)

## Responsibilities

- Admin dashboard with analytics
- User management (block/unblock, view details)
- Female approval workflow
- Coin economy management (plans, payout slabs)
- Withdrawal management
- Transaction history
- Audit logs
- Settings management
