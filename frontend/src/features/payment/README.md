# Payment Feature Module

Coin purchase and payment integration.

## Structure

- `components/` - Payment UI (CoinPlanCard, PaymentForm, etc.)
- `hooks/` - Payment hooks (usePayment, useRazorpay, etc.)
- `services/` - Payment API calls and Razorpay integration
- `types/` - Payment types (Payment, CoinPlan, RazorpayOrder, etc.)

## Responsibilities

- Coin plan selection
- Razorpay payment integration
- Payment verification
- Payment success/failure handling
- Order creation and management
