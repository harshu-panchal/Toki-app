# Wallet Feature Module

Wallet and coin economy management.

## Structure

- `components/` - Wallet UI (WalletBalance, TransactionList, CoinPlanCard)
- `hooks/` - Wallet hooks (useWallet, useTransactions, etc.)
- `services/` - Wallet API calls
- `types/` - Wallet types (Wallet, Transaction, CoinPlan, etc.)
- `store/` - Zustand store for wallet state

## Responsibilities

- Wallet balance display
- Transaction history
- Coin tier management (Basic/Silver/Gold/Platinum)
- Transaction filtering and display
- Balance updates
