# Auth Feature Module

Authentication and authorization functionality.

## Structure

- `components/` - Auth-specific UI components (LoginForm, RegisterForm, etc.)
- `hooks/` - Custom hooks (useAuth, useLogin, useRegister, etc.)
- `services/` - API calls for authentication
- `types/` - TypeScript types for auth (User, LoginRequest, etc.)
- `pages/` - Page components (LoginPage, RegisterPage, OTPVerificationPage)

## Responsibilities

- User registration (male/female role selection)
- User login/logout
- JWT token management
- OTP verification (optional)
- Password reset
- Session management
