# Profile Feature Module

User profile management.

## Structure

- `components/` - Profile UI (ProfileCard, ProfileForm, PhotoUpload)
- `hooks/` - Profile hooks (useProfile, useUpdateProfile, etc.)
- `services/` - Profile API calls
- `types/` - Profile types (Profile, ProfileUpdate, etc.)
- `store/` - Zustand store for profile state

## Responsibilities

- Profile CRUD operations
- Photo upload and management
- Profile completion validation
- Preferences management (for males)
- Profile viewing
